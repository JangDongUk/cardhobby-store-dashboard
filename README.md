# 카드하비 스마트스토어 대시보드

스마트스토어 매출/마케팅 데이터를 시각화한 대시보드

🔗 [대시보드 바로가기](https://jangdonguk.github.io/cardhobby-store-dashboard/dashboard.html)

## 기능

- 일/주/월/년 기간별 매출 조회
- 현재 조회 기간 상시 표시
- 매출 트렌드 막대 클릭 → 해당 날짜 상세 데이터로 이동
- 라이트 / 다크 모드 선택 (선택값 자동 저장)
- 카테고리·브랜드·시리즈별 분석
- 채널별 유입 분석
- 키워드 성과
- 광고 ROAS 추적
- 요일별 패턴 분석
- 운영 메모 타임라인
